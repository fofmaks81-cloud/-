# Сайт SOP (стоматология)

Локальный статический сайт с интерактивным выбором протоколов.

## Что уже сделано
- Раздел `Ортопедия` (список протоколов подгружается при открытии)
- Просмотр выбранного протокола (HTML вставляется в контейнер)

## Как добавить ваши протоколы
1. Пришлите HTML-коды ваших протоколов ортопедического лечения.
2. Для каждого протокола укажите заголовок (как он должен отображаться в выпадающем списке).

После этого я:
- удалю шаблонный пример из `data/ortopedia.json`
- добавлю ваши протоколы так, чтобы они открывались интерактивно по клику в разделе `Ортопедия`.
